HTML Practice:

HTML is the acronym of Hyper Text Markup Language, which is a language for specifying the content and structure of a web page

In HTML5, the beginning of an HTML document should be:
<!DOCTYPE HTML>

Four Elements in the head:
<TITLE>: gives a title and maximizes the find-ability of the page by search engines
<STYLE>: defines an internal CSS sheet
<LINK>: links this page with an external CSS document
<META>: gives information about this page so it maximizes it's find-ability by search engines


Making an Internal link of a web page:
Create a point by giving a value to the id attribute in an element

<P id="top" > Home </P>
Use an anchor and it's href attribute to reference this point

<A href='#top'>Click to go back to home</A>
