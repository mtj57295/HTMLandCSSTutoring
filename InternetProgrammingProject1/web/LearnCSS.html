"Cascading Style Sheet"
CSS Statement Syntax:

property_name:value;
property name value
font-family arial
color red, yellow, or color code
font-size: 22px, 2em, 150%, 2in

Three Levels:

1. inline -- place css statements as the value of the style attribute of a tag
2. embedded -- place css statements into selectors and place the selectors into the page as the content of the style tag
3. external document -- place css statements into selectors. Place the selectors into an external document with postfix .CSS and link your page to the doc. by using the href attribute of the style tag

Three CSS Selectors:

1. HTML selector: styles the contents of the tags with the given name
syntax: htmlTagNames{scc_Statements}
2. Class Selector: it can be applied to the content of a tag if you place the class name as the value of the class attribute
3. ID Selector

Pseudo-class selectors:

:className{css_statements }
:hover-- used to style the content of an element the mouse is hovering over things
:active-- used to select and style the active elements. An element is active if you click on it and hold it
:link-- used to style unvisited links
:visited-- used to style links that have been visited
:first-child-- select the element that is the first child (in nest) of its parent
::first-letter -- selects the first letter of this element
::before -- insert content before the content of the element
::after -- insert content after this element

Group Several Selectors:

.headline, .subline{font-size:50px}

Context Dependant Selectors:

B I{font-size:20pt; color:green;} if B, I then it shares attributes
- the properties will be applied to the I tag as a child of the B tag

Combination of group selectors and content dependant selectors:

I B, B .test2{font-size:75px;}
1.) The property is applied to tag B when it is a child of an I tag
2.) The property is applied to a test2 class tag when it is a child of B tag

CSS Practice:

CSS and what it is for:
CSS is the acronym of "Cascading Style Sheet" which is used to style a web page

General syntax of a CSS Statement:
Property:value;

Three kinds of selectors in CSS:
1.) HTML Selector:
syntax - HTMLElementName{ cssStatements }
2.) Class Selector:
syntax - ClassName{ cssStatements }
3.) ID Selector:
syntax - #IDValue{ cssStatements }

Three levels of CSS:
1.) Inline styles: uses the HTML style attribute to declare an individual style and overrides any other styles related to element so inline has highest priority

2.) Embedded style sheet: places a CSS document in an HTML document's head section

3.) External Style Sheet: saves a CSS document in a file with postfix .css and links is to this page by using the link element in the head section

Difference between <DIV> and the <SPAN>:
both are dummy elements for holding attributes. <SPAN> is an inline element which does not change line.
<DIV> is a block element which changes line before and after the content of this element

 CSS Pseudo class that renders the content of a div tag in red color when it is the first child of the parent tag:
div:first-child{color:red}
